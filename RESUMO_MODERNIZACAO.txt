🚀 MELHORIAS TÉCNICAS
═════════════════════════════════════════════════════════════════════════════

1. SCROLL ANIMATIONS
   ├─ Intersection Observer (nativo, sem lib)
   ├─ Elemento fade + slide ao entrar viewport
   └─ Timing perfecto: não brusco, não slow

2. HOVER EFFECTS
   ├─ Buttons: translateY(-3px) + shadow increased
   ├─ Cards: Border color change + lift up
   ├─ Links: Underline animation from left
   └─ Gallery: Scale 1.08 + overlay gold

3. PARALLAX SUAVE
   ├─ Mouse follow (hero background)
   ├─ Não é invasivo
   ├─ Cria sensação de depth
   └─ Desktop only (performance)

4. MENU MOBILE POLIDO
   ├─ Hamburger se transforma em X
   ├─ Menu slide down com blur
   ├─ Items com stagger (50ms cada)
   └─ Fechar ao clicar

5. HEADER INTELIGENTE
   ├─ Blur aumenta com scroll
   ├─ Hide quando scroll down (UX moderna)
   ├─ Reappear quando scroll up
   └─ Sombra dinâmica
