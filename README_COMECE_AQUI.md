## 🐛 TROUBLESHOOTING

### Problema: Animações não funcionam
**Solução:** Verificar console (F12) por erros de JS

### Problema: Muito lento em mobile
**Solução:** Desabilitar efeitos pesados:
```javascript
if (!isTouchDevice()) {
  // Ativar efeitos avançados apenas desktop
}
```

### Problema: Acessibilidade
**Solução:** Incluído automaticamente `prefers-reduced-motion`
